# student-performance-analysis

This project focuses on analyzing student academic performance using python. The gaol is to identify patterns, trends and factors affecting student scores.

Source : Student Performance Dataset 
Format : CSV

Python

Visualizations using charts and graphs

Exploratory Data Analysis
Visual charts saved as HTML
